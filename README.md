# PEMS (Project & Workforce Management System)

A SaaS system for internal resource management, project tracking, and team time tracking.

## 🛠 Tech Stack

* **Backend:** Node.js, Express, TypeScript, Prisma ORM, PostgreSQL (Neon), JWT, Swagger (OpenAPI)
* **Frontend:** React Router 7 (SPA / SSR)
* **Database:** PostgreSQL

---

## 📌 Core Features (MVP)

* **Authentication & Roles:** Role-based access control for `Owner`, `Manager`, and `Employee`.
* **Project Management:** Project creation and tracking across lifecycle stages (*Planned / Active / Done*).
* **Workforce Management:** Assigning personnel to relevant projects.
* **Time Tracking:** Logging working hours by employees.
* **Payroll:** Basic wage calculation based on tracked hours and hourly rates.
